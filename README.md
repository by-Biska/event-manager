# Event Manager API 🎫

Микросервис для управления мероприятиями и бронирования билетов.

## 📋 О проекте

Простой REST API сервис, который позволяет:
- Регистрироваться пользователям
- Создавать мероприятия
- Бронировать билеты
- Просматривать свои бронирования

## 🚀 Технологии

- **Python 3.11** - язык программирования
- **FastAPI** - фреймворк для API
- **PostgreSQL** - база данных
- **SQLAlchemy** - работа с БД
- **Docker** - контейнеризация
- **JWT** - аутентификация
